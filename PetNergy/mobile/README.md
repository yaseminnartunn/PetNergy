# PetNergy Mobile (Expo)

Bu klasör, PetNergy’nin **Expo SDK 54** ile çalışan mobil uygulamasıdır.

## Çalıştırma

```bash
cd mobile
npm install
npx expo start -c
```

Telefon:
- Expo Go → **Scan QR code** ile QR kodu okut.

## Notlar

- Expo SDK: `~54.0.0`
- Paket sürümleri Expo ile hizalıdır.
- Veri akışı demo amaçlıdır (pet state / log akışı).
